# 🎬 CineVault

> Твоя персональна кінотека — зберігай, оцінюй та ділись улюбленим кіно.

## 📁 Структура проєкту

```
cinevault/
├── index.html          # Головна сторінка
├── css/
│   └── style.css       # Всі стилі
├── js/
│   └── main.js         # Базова інтерактивність
├── images/             # Зображення (постери, логотип тощо)
├── fonts/              # Локальні шрифти (якщо є)
└── README.md
```

## 🚀 Запуск

Відкрити `index.html` у браузері або запустити локальний сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .
```

## 🎨 Технології

- **HTML5** — семантична розмітка (`<header>`, `<main>`, `<section>`, `<article>`, `<footer>`, `<nav>`, `<figure>`, `<aside>`)
- **CSS3** — змінні, Grid, Flexbox, анімації, адаптивний дизайн
- **Vanilla JS** — мобільне меню, мікро-взаємодії

## 🖌️ Дизайн-система

| Токен | Значення |
|-------|---------|
| `--color-bg` | `#0D0D1A` — основний фон |
| `--color-accent` | `#E5A817` — золотистий акцент |
| `--color-teal` | `#0AB3B8` — бірюзовий |
| `--font-display` | Cinzel (заголовки) |
| `--font-body` | Nunito Sans (текст) |

## ✅ Семантичні теги

- `<header>` — шапка сайту
- `<nav>` — навігаційні блоки
- `<main>` — основний контент
- `<section>` — тематичні секції
- `<article>` — картки фільмів
- `<aside>` — постер у hero
- `<figure>` / `<figcaption>` — медіа з підписом
- `<footer>` — підвал сайту
